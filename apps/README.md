# Apps

This directory contains LLMS.txt files for applications and tools.

## Contributing

When adding a new app LLMS.txt file:

1. Name the file as `app-name-llms.txt`
2. Include:
   - Overview of the application
   - Key features and capabilities
   - Installation and setup
   - Common usage patterns and commands
   - Configuration options
   - Links to official documentation
3. Focus on practical information that helps LLMs understand how to use the application
