# Services

This directory contains LLMS.txt files for cloud services and platforms.

## Contributing

When adding a new service LLMS.txt file:

1. Name the file as `service-name-llms.txt`
2. Include:
   - Overview of the service
   - Key features and capabilities
   - Authentication and setup
   - Common usage patterns with code examples
   - Pricing/tier information (if relevant)
   - Links to official documentation
3. Focus on practical information that helps LLMs understand how to use the service
