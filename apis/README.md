# APIs

This directory contains LLMS.txt files for various web APIs and programming interfaces.

## Available APIs

- **WebGPU** - Modern GPU API for the web

## Contributing

When adding a new API LLMS.txt file:

1. Name the file as `api-name-llms.txt` (e.g., `webgpu-llms.txt`)
2. Include:
   - Overview of the API
   - Key concepts and terminology
   - Common usage patterns with code examples
   - Available methods/endpoints
   - Links to official documentation
3. Focus on practical information that helps LLMs understand how to use the API
